# Contact API

> Rest API for manage and make available an agenda of people's contacts

## Requirements:

- Endpoints to:
    - [ ] register
    - [ ] edition
    - [ ] deletion
    - [ ] list

- Entities:
    - Contact:
        - id
        - first name
        - last name
        - email
        - telephones (many)
        > All fields are required

- Specifications:
    - [ ] Contact must have at least 1 phone.
    - [ ] In listing, it must be possible to filter by name or email.
